Top 3 Steps to Make Homemade Pizza

  1. Roll out the pizza dough onto a baking sheet.
  2. Spread tomato sauce evenly and add cheese and toppings.
  3. Bake in a preheated oven at 220°C for 12 to 15 minutes.

HTML Glossary

HTML
HyperText Markup Language, the standard language used to create web pages.
Element
An individual component of an HTML document, consisting of a start tag, content, and an end tag.
Attribute
Special words used inside the opening tag to control the element's behaviour or provide extra information.
